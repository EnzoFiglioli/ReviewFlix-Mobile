# Aplicacion movil de Reseñas de peliculas
ReviewFlix nace en 2023 como version web para un proyecto de Codo a Codo como trabajo final donde las practica era hacer un proyecto para realizar un crud basico. 

Mi objetivo fue expandir y realizar una app que pueda puntuar y hacer comentarios.  

Hoy en dia, como practica para la materia `Desarrollo de aplicaciones moviles`decidi emprender el proyecto para tener una version Mobile de la aplicación.
